Simple restaurant API documentation version 1

http://restaurant-api.devcharles.com

Description

Restaurant API is a simple API that provide a simple interface to manage orders in a restaurant

If you want to give a try to this API please visite the postman collection here, where you can download the collection and run it in your local postman or just see some request examples in diferent languajes like NodeJS, Go, Python or using cURL command

Hope you like it!

/orders/{orderId}/dish/{dishId} patch delete

patch

Add the given dish into the given order

delete

Remove given dish of the given order

patch /orders/{orderId}/dish/{dishId}

Add the given dish into the given order

  • Request
  • Response

URI Parameters

  • orderId: required (string)

    The order id as it is in mongo

  • dishId: required (string)

    The dish id as it is in mongo

HTTP status code 200
